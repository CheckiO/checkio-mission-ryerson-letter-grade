You just need to wrap your code into function expression and return letter grade.

Here is the whole solution:

LETTERS = [ch + sign for ch in 'ABCD' for sign in ('+','','-')] + ['F']
MIN_PERCENTAGES = 90, 85, 80, 77, 73, 70, 67, 63, 60, 57, 53, 50, 0

def ryerson_letter_grade(pct: int) -> str:
    
    i = 0
    while MIN_PERCENTAGES[i] > pct:
        i += 1
    
    return LETTERS[i]

GRADE = ['F']*50 + ['D-']*3 + ['D']*4  + ['D+']*3 + ['C-']*3 + \
['C']*4  + ['C+']*3 + ['B-']*3 + ['B']*4  + ['B+']*3 + \

ryerson_letter_grade = lambda percent: GRADE[min(percent, 100)]

In this solution grades are in the single string which is splitted by whitespaces. The grade list and percentage tuple are the arguments of zip() function - which takes values with the same index. 

def ryerson_letter_grade(pct: int) - str:

    for letter_grade, percentage in zip(
        'A+  A   A-  B+  B   B-  C+  C   C-  D+  D   D-  F'.split(),
        (90, 85, 80, 77, 73, 70, 67, 63, 60, 57, 53, 50, 0)):
        if pct >= percentage:

            return letter_grade
